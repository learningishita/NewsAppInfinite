# News App - InfiniteScroll

The news app has infinite scrolling where it fetches the news from PinkVilla.

### Technology Stack
MERN

## Project Setup

```sh
git clone https://github.com/learningishita/NewsAppInfinite.git
cd NewsAppInfinite
```

### Backend

```sh
cd backend
npm install
node server.js
```


### Frontend

```sh
cd frontend
npm install
npm start
```
